# Reconciliation

## Overview
Reconciliation tools help you verify that the supply of bridged assets on Stellar exactly matches the locked assets on their native chains.

## How it Works
1. Navigate to the **Reconciliation** page.
2. Select an asset and its corresponding bridge.
3. The platform will query both the Stellar network and the native chain (e.g., Ethereum).
4. The system compares the total minted tokens on Stellar with the total locked tokens in the native smart contract.

## Resolving Discrepancies
If a discrepancy is detected, the system provides a detailed breakdown of recent transactions to help you identify the missing or extra mints/locks.
