# Alerts

## Overview
The Alerts feature allows you to monitor critical changes in asset health, price deviations, and bridge status across the Stellar network.

## Configuring Alerts
1. Navigate to the **Alerts** tab on the left sidebar.
2. Click **Create Alert**.
3. Choose the alert type:
   - **Price Deviation**: Triggers when price differences exceed X%.
   - **Supply Mismatch**: Triggers if locked vs minted supplies diverge.
   - **Circuit Breaker**: Triggers on sudden liquidity drops.
4. Set your custom threshold values.
5. Select the notification channels (Email, Webhook, Discord).
6. Click **Save Alert**.

## Managing Alerts
You can view, edit, or toggle the active status of any existing alert from the Alerts Dashboard.
