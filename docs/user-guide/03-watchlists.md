# Watchlists

## Overview
Watchlists allow you to create customized groups of assets, grouping them by ecosystem, risk level, or any metric relevant to your workflow.

## Creating a Watchlist
1. Navigate to the **Watchlists** page.
2. Click **Create New Watchlist**.
3. Provide a name and description for your watchlist.
4. Search for assets and bridges to add to your list.
5. Click **Save Watchlist**.

## Viewing and Filtering
Once your watchlist is active, you can apply it as a global filter across the Dashboard and Analytics pages.
